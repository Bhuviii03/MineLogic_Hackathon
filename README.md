# SafeMine — Agentic AI for Coal Mine Safety in India

SafeMine is an AI-powered mine safety intelligence platform that transforms DGMS coal mine incident reports into **actionable insights**, **trend alerts**, and **automated safety audit reports**, supported by an interactive **Digital Mine Safety Officer**.

>  Focus: Coal Mines in India • DGMS Records (2016–2022)

---

## Problem

Mining safety decisions in India often rely on manual review of PDF reports, which is:
- Time-consuming
- Error-prone
- Lacks trend detection ✅

SafeMine extracts hidden intelligence from historical data to **prevent future accidents**.

---

## Key Features

| Feature | Capability |
|--------|------------|
| PDF Upload & Auto-Extraction | Extracts date, state, mine type, cause, severity |
| Conversational Query Agent | e.g., “Gas leak underground Jharkhand 2021” |
| Autonomous Trend Watcher | Detects spikes in severity & accident frequency |
| Auto Safety Audit Reports | Export KPIs grouped by any filter |
| Visual Analytics | Heatmaps, Risk Priority Matrix, Age-Risk charts |
| Modular & Expandable | Can integrate compliance & IoT alerts |
